**Проект: "Unchain" – Android-приложение для отслеживания привычек и борьбы с зависимостями.**

**Технологии:** Kotlin, Android SDK, MVVM, Clean Architecture, Coroutines, Flow, Room, WorkManager, Dagger 2, AppWidget, BroadcastReceiver, ViewBinding, Retrofit

---

## Что реализовано:

* Разработка приложения с использованием **Clean Architecture** (разделение на Domain, Data и Presentation слои).

* Реализована система отслеживания прогресса пользователя:

  * текущий и лучший streak
  * ежедневные отметки (успех/провал)
  * внутренняя “валюта” за прогресс

* Использование **Room** для хранения данных и реактивного обновления UI через **Flow**.

* Реализована бизнес-логика через **UseCase**, обеспечивающая чистую и тестируемую архитектуру.

* Добавлены напоминания с помощью **WorkManager** и планирования задач по времени.

* Реализован **Home Screen Widget**:

  * отображение выбранной привычки и текущего прогресса
  * возможность взаимодействия (отметка успешного/неуспешного дня прямо из виджета)
  * обновление данных через **BroadcastReceiver**

* Настроено внедрение зависимостей с помощью **Dagger 2**.

* Работа с асинхронностью через **Kotlin Coroutines**.

* Обработка жизненного цикла UI с использованием **ViewModel и Flow**.

* Использование **Retrofit** для получение мотивации от Gemini
